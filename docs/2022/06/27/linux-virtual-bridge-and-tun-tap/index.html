在这种类型的连接中，VM 被插入 - 有点 - 在连接主机的同一交换机上。例如，如果网络上有 DHCP 服务器并且主机从该 DHCP 获取其 IP，则 VM 也将能够从同一 DHCP 获取 IP。这是主要用于生产 ESXi、Xen 或 HyperV 虚拟机的场景。

警告 当物理网卡分配给网桥时，它会失去连接。IP必须分配给网桥，而不是主机的物理网络接口。
